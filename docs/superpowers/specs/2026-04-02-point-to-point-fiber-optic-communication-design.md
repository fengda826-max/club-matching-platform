# 点到点光纤通信系统设计 (1 Gb/s, 100 km)

## 系统规格

| 参数 | 规格 | 说明 |
|------|------|------|
| 比特率 | **1 Gb/s** | NRZ编码 |
| 传输距离 | **100 km** | 单跨传输 |
| 调制格式 | **NRZ (不归零码)** | 二进制强度调制 |
| 光纤类型 | **G.652 标准单模光纤** | 1550 nm 低损耗窗口 |
| 光源类型 | **DFB 分布式反馈激光器** | 直接调制，1550 nm |
| 功率放大 | **EDFA 掺铒光纤放大器** | 后置功率放大 |
| 接收方式 | **相干检测** | 高灵敏度 |
| 工作波长 | **1550 nm** | 最低损耗 |

## 系统架构

```
[伪随机码发生器] → [NRZ编码器] → [DFB激光器] → [EDFA] → [G.652光纤 100km] → [相干接收机] → [BER分析仪]
```

## 器件参数设计

### 1. 发射端

**伪随机码发生器 (PRBS):**
- 速率：1 Gb/s
- 序列长度：2^31 - 1
- 输出电平：0 ~ Vpi (适配调制器驱动)

**DFB激光器：**
- 中心波长：1550.0 nm
- 线宽：10 MHz
- 输出功率：0 dBm
- 相对强度噪声(RIN)：-155 dB/Hz
- 啁啾参数：0.05 nm/mW

**EDFA 掺铒光纤放大器：**
- 小信号增益：15 dB
- 输出功率：15 dBm
- 噪声指数：5.0 dB
- 增益平坦度：±0.5 dB

### 2. 光纤传输链路

**G.652 单模光纤：**
- 长度：100 km
- 衰减系数：0.20 dB/km @ 1550 nm
- 色散系数：17 ps/(nm·km) @ 1550 nm
- 色散斜率：0.075 ps/(nm²·km)
- 有效面积：80 μm²
- 非线性折射率：2.6 × 10^-20 m²/W

**总链路损耗：** 0.20 × 100 + 2 (连接器) = 22 dB

**输出功率预算：** 15 dBm (EDFA输出) - 22 dB (链路损耗) = -7 dBm (接收端输入功率)

### 3. 接收端 (相干检测)

**本振激光器 (LO)：**
- 类型：DFB激光器
- 中心波长：1550.0 nm
- 线宽：10 MHz
- 输出功率：10 dBm

**90°光混频器：**
- 插入损耗：3 dB
- 相位误差：< 2°

**平衡光电探测器：**
- 响应度：0.9 A/W @ 1550 nm
- 暗电流：10 nA
- 热噪声密度：10 pA/√Hz

**DSP处理：**
- 时钟恢复
- 载波相位估计
- 偏振分集处理
- 误码率计算

## 参数预算

| 项目 | 值 |
|------|-----|
| EDFA输出功率 | +15 dBm |
| 光纤损耗 (0.2 dB/km × 100 km) | -20 dB |
| 连接器损耗 (2个) | -2 dB |
| 光混频器插入损耗 | -3 dB |
| 接收光功率 | **-10 dBm** |
| 相干接收灵敏度 | ~-30 dBm @ 1e-9 BER |
| **功率余量** | **+20 dB** |

功率预算充足，系统留有很大余量。

## 预期性能

- **接收灵敏度**：优于 -25 dBm @ BER = 10^-9
- **Q因子**：预计 > 15 dB
- **误码率**：BER < 10^-12 室温工作

## 布局步骤 (在OptiSystem中搭建)

1. 从元件库拖放组件到工作区
2. 按信号流向连接各组件
3. 设置各组件参数如上
4. 添加观测仪（眼图分析仪、光谱分析仪、BER分析仪）
5. 运行仿真，查看结果

## 观测仪器配置

- **光谱分析仪**：观测发射端和接收端光谱
- **眼图分析仪**：观测接收信号眼图
- **误码率分析仪**：测量BER和Q因子
- **光功率计**：各点功率检测
- **示波器**：观测电信号波形
