## 内容规范

| 部分 | 规范 |
|------|------|
| **基本原理** | 中文阐述，简洁（≤300字/实验），重点清晰 |
| **命令表格** | 三列格式，只包含实验中实际用到的命令 |
| **实验结果** | 完整展示所有命令及其在蓝桥环境中的实际输出 |
| **格式** | 标准 GitHub Markdown，代码块用 ```bash 高亮 |
